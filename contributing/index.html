How to contribute

TL;DR: Fork the bna-tech-companies repo on Github, add your company, and submit a pull request.

The instructions

Adding a company to the list

To add your company (or a company you know of; feel free to add any company that's not already on the list):

  1. Fork the bna-tech-companies repo on Github.
  2. In /_data/companies, pick a JSON file to use as a template (for example, isis.json). Create a copy of that file in the same directory, with a name reflecting your company.
  3. Enter your company's information into each field in the JSON file. Take note of the "locations" field: this is an array of ZIP codes and can contain more than one (if you have multiple offices or locations).
  4. Commit your new JSON file, and push to your fork at Github.
  5. Open a pull request at {{site.github_repository}}.
  6. That's it! If your information is accurate, your pull request should be merged promptly and your company will appear in the list as soon as Github Pages updates.
